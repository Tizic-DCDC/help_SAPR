# help_SAPR
Средство для тедекции человеческой речи с последующей пеленгом.  Не самый оптимальный формат работы но со своей целью справляется.
Компанентая часть
  1) 3 микрофона max9814,
  2) SMT32F401CCU6,
  3) rasberry pi 5,
  4) модуль на XL6019,
  5) акб типа 18650 в параллельном подключении.
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
Аппаратная часть для SMT32F401CCU6
  1) DMA кольцевой,
  2) АЦП 3 канала с (не помню дополнить) с DMA,
  3) UART для связи с RP 5 с DMA  STM ->TX->RX<-RP5.
  Аппаратная часть для Rasberry pi 5
  1) RP5 использовал uart
